# 불한당들 — Scoundrels

> Pronounced "Bool-han-dang-deul"
> 1인 - 4인 협력 로그라이크 카드 게임 — Zach Gage & Kurt Bieg 원작 Scoundrel 기반
> PWA 버전 (오프라인 지원 · 모바일 최적화 · 다크모드 지원)

---

## 게임 소개

불한당들은 일반 트럼프 카드를 기반으로 한 로그라이크 카드 게임입니다. 던전(구역)을 탐험하며 몬스터를 처치하고, 무기와 포션을 활용해 살아남는 것이 목표입니다.

이 버전은 원작 Scoundrel(Zach Gage & Kurt Bieg, 2011)을 기반으로 아래 기능을 추가한 PWA(Progressive Web App)입니다.

- 1인 / 2-4인 협력 모드
- 배경 설정 입력 (판타지, 우주선, 폐건물, 연구소 등 자유 설정)
- 덱 현황 표 (처리된 카드 실시간 추적, 설정에서 켜고 끄기 가능)
- 턴별 플레이 기록 자동 생성 → 클립보드 복사 / `.md` 다운로드
- AI를 활용한 단편 소설 생성 연동 가능

---

## 카드 구성

총 44장으로 플레이합니다. (조커, 빨간색 J/Q/K/A 제거)

| 카드 | 수량 | 역할 |
|---|---|---|
| ♣ 클럽 / ♠ 스페이드 (2-A) | 26장 | 몬스터 — 수치만큼 에너지 피해 |
| ♦ 다이아몬드 (2-10) | 9장 | 무기 — 피해 경감 |
| ♥ 하트 (2-10) | 9장 | 회복 포션 — 에너지 회복 |

**카드 수치 환산:** J=11, Q=12, K=13, A=14

---

## 기본 룰

### 진행

1. 매 턴 던전에서 카드 4장을 뒤집어 **구역**을 구성합니다.
2. 4장 중 **3장을 선택해 처리**하고, 나머지 1장은 다음 구역으로 이월됩니다.
3. 구역 전체를 **우회**할 수 있지만, 연속으로 두 번은 불가능합니다.

**다인 플레이에서의 우회:**
한 플레이어가 구역을 우회하면 해당 카드들은 던전 맨 아래로 밀리고 턴이 다음 플레이어에게 넘어갑니다. 이때 다음 플레이어는 반드시 새 구역을 탐색해야 하며 연속 우회는 불가능합니다. 우회를 반복하면 강한 카드들이 후반에 몰려 오히려 불리해질 수 있습니다.

### 전투

- **맨손**: 몬스터 에너지만큼 피해를 받습니다.
- **무기 사용**: `몬스터 에너지 - 무기 수치 = 받는 피해` (0 미만이면 피해 없음)

### 무기 사용 제한 (핵심 전략)

무기로 몬스터를 처치한 이후에는, **처치한 몬스터 에너지 이하의 몬스터에게만** 해당 무기를 다시 사용할 수 있습니다. 무기 수치 자체는 변하지 않습니다.

```
예: 무기(5)로 몬스터(8) 처치 → 이후 ≤8 몬스터에게만 무기 사용 가능
예: 무기(5)로 몬스터(3) 처치 → 이후 ≤3 몬스터에게만 무기 사용 가능 (불리!)
```

**전략 포인트:** 강한 몬스터부터 무기로 처리해야 사용 가능 범위가 넓게 유지됩니다.

**다인 플레이에서의 무기 전달:**
자신의 턴에 다음 순서 플레이어에게만 무기를 넘길 수 있습니다. 이때 사용 제한(`≤N까지`)도 함께 이어집니다. 예를 들어 A가 무기(5)로 몬스터(8)을 처치한 뒤 B에게 무기를 넘기면, B도 해당 무기로 ≤8 몬스터에게만 사용할 수 있습니다. 무기를 넘긴 플레이어는 이후 맨손 전투만 가능합니다. 3-4인 플레이에서 특정 플레이어에게 전달하려면 중간 플레이어를 거쳐 릴레이해야 합니다.

### 포션

- 수치만큼 에너지를 회복합니다.
- 에너지는 20을 초과할 수 없습니다.
- 한 턴에 하나만 사용 가능합니다. 두 번째 포션은 버려집니다.

### 종료 및 점수

- 에너지가 0이 되면 **탈진(실패)**합니다.
- 던전 전체를 통과하면 **탈출 성공**입니다.
- **성공 시**: 생존자 에너지 합산이 점수
- **실패 시**: 남은 몬스터 에너지 합산만큼 패널티 (음수 점수)

## UI 안내

### 플레이어 패널

각 플레이어 카드 상단에 무기 뱃지가 표시됩니다.

| 상태 | 표시 예시 | 의미 |
|---|---|---|
| 무기 없음 | `♦ 없음` | 맨손 전투만 가능 |
| 무기 장착, 제한 없음 | `♦ ♦7 · ≤∞까지` | 모든 몬스터에 무기 사용 가능 |
| 무기 장착, 제한 있음 | `♦ ♦7 · ≤5까지` | 에너지 5 이하 몬스터에만 사용 가능 |

### 몬스터 카드 뱃지

무기를 장착한 상태에서 구역을 탐색하면, 각 몬스터 카드 하단에 무기 사용 가능 여부가 자동으로 표시됩니다.

| 뱃지 | 의미 |
|---|---|
| `♦ 피해 3` (파란색) | 무기 사용 가능, 피해 3 |
| `♦ 사용불가` (빨간색) | 무기 사용 불가, 맨손 전투 필요 |

### 전투 선택 팝업

몬스터 카드를 탭하면 전투 방식을 선택하는 팝업이 열립니다.

- 현재 장착 무기 정보와 사용 제한 수치 표시
- 무기로 싸울 경우 이후 제한이 어떻게 변하는지 미리 표시
- 무기 사용 불가 시 이유 명시

### 덱 현황 표

게임 화면 방 카드 아래에 4행 × 13열 표가 고정 표시됩니다. (설정에서 끄기 가능)

- ✓ (녹색): 이미 처리된 카드
- 빈칸: 아직 던전에 남은 카드
- · : 이 게임에 포함되지 않는 카드 (하트/다이아 J/Q/K/A)

## 협력 모드 (2-4인)

- 플레이어는 턴을 돌아가며 같은 던전을 탐험합니다.
- 각자 에너지 20을 보유하며, **한 명이라도 에너지 0이 되면 전원 실패**합니다.
- 자신의 턴에 다음 플레이어에게 무기를 넘길 수 있습니다.
- 무기를 넘길 때 사용 제한(`≤N까지`)도 함께 이어집니다 (원작 룰).
- 한 플레이어가 우회하면 다음 플레이어는 반드시 해당 구역을 처리해야 합니다.

## 배경 설정

게임 시작 전 세계관을 설정할 수 있습니다. 프리셋 4가지를 제공하며 자유롭게 수정 가능합니다.

| 프리셋 | 설명 |
|---|---|
| ⚔️ 판타지 | 중세 석조 던전을 탐험하는 모험가 |
| 🚀 우주선 | 표류하는 화물선의 이상 구역 조사 |
| 🏚️ 폐건물 | 도심 외곽 공장 지하 실종자 수색 |
| 🧪 연구소 | 봉쇄된 바이오 연구소 탈출 |

## 플레이 기록 & 스토리 생성

게임 종료 시 마크다운 형식의 플레이 기록이 자동 생성됩니다.

```markdown
# 불한당들 — 던전 탐험 기록

## 배경 설정
중세 판타지 세계. 고대 석조 던전 속...

## 게임 정보
- 플레이어: 철수, 영희 (2인 협력)
- 결과: 탈출 성공
- 최종 점수: +27

## 턴별 기록
### 턴 1 — 철수
- 방 구성: ♣7(몬스터 7), ♦5(무기 5), ♥3(포션 3), ♠Q(몬스터 12)
- 무기 장착: ♦5(무기 5)
- 회복: ♥3(포션 3) → 체력 +3 → 20HP
- 무기 전투: ♣7(몬스터 7) vs 무기5 → 피해 2 → 체력 18HP
- 이월 카드: ♠Q(몬스터 12)
```

기록을 Claude에 붙여넣고 **"이 로그를 바탕으로 던전 탐험 단편소설을 써줘"** 라고 하면 플레이 내용이 소설로 각색됩니다.
